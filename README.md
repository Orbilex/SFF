# 🤖 Servo: Frame Field

**A high-complexity tactical survival experiment built to push the limits of performance and logic.**

`Servo: Frame Field` is a high-intensity tower defense game. It explores how deep and mechanically complex a web-based experience can get, focusing on high entity counts and reactive game states.

## ⚡ Technical Highlights
This project serves as a stress test for modern web performance, featuring:
* **High-Density Swarm AI:** Advanced pathing logic that allows hundreds of units to adapt to your perimeter in real-time.
* **Optimized Frame Logic:** High-precision interactions between towers (Servos) and the field, utilizing object pooling to maintain stability.
* **Deep Tech Trees:** Multiple upgrade paths that dynamically alter gameplay loops and projectile physics.
* **Hardware-Aware UI:** A reactive visual style designed to feel like a living machine while remaining performant across devices.
